# API-часть

## 1. Как должен выглядеть запрос на отправку текстового сообщения

Эндпоинт для отправки сообщения:

`PUT /_matrix/client/v3/rooms/{roomId}/send/{eventType}/{txnId}`

Для обычного текстового сообщения я ожидаю:
- `roomId` - идентификатор комнаты;
- `eventType = m.room.message`;
- `txnId` - уникальный идентификатор транзакции, который клиент использует для идемпотентности.

### Ожидаемый body запроса

Минимально я ожидаю в body:
- тип сообщения;
- текст сообщения.

Пример минимального body:

```json
{
  "msgtype": "m.text",
  "body": "Hello QA"
}
